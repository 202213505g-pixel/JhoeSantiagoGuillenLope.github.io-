Ejercicio 1: Tarjetas

Esto es una tarjeta 1. El modelo de caja CSS define el espacio que ocupa cada elemento en la página.

Esto es una tarjeta 2. El padding agrega espacio interno entre el contenido y el borde del elemento.

Esto es una tarjeta 3. El margin crea espacio externo separando el elemento de sus vecinos.

Ejercicio 2: Caja de Contenido

El box model es un concepto fundamental de CSS. Cada elemento HTML se representa como una caja rectangular con cuatro capas: contenido, relleno, borde y margen.

Usar propiedades individuales como margin-top, margin-right, margin-bottom y margin-left permite un control preciso sobre cada lado del elemento.
